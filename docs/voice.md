# Aura voice — the spec

The brand voice in one place. If you change copy and you're not sure if it's on-voice, run it through this doc.

---

## The five rules

1. **Lowercase, except proper nouns and the start of headlines.** Mono labels are uppercase.
2. **One italic phrase per page.** A soft beat — `let's begin`, `still listening`, `just a moment`. Never two.
3. **No exclamation marks.** Anywhere. Ever.
4. **Short sentences.** A long one earns its length only by saying something true.
5. **Never over-promise.** "we'll try" beats "we will". Quiet confidence > loud claims.

## The banned words

These signal AI slop or generic SaaS marketing. Don't use them:

| Banned | Reach for instead |
| --- | --- |
| Unleash | open, start |
| Transform | change, rework |
| Supercharge | speed up, sharpen |
| Revolutionize | rethink, change |
| Empower | help, give |
| Leverage | use |

## The italic phrases (the soft beats)

You get **one per page**. Pick from these or invent in the same register:

- `let's begin`
- `still listening`
- `just a moment`
- `not quite yet`
- `a small thing, carefully made`
- `quietly`
- `softly`
- `here, then`
- `something new`
- `nothing yet`
- `just begun`

Anti-examples (do not use): `magical`, `delightful`, `amazing`, `effortless`, `seamless`.

---

## The headline shapes

Three shapes work. Pick one per page:

**1. Statement, soft beat**
> a small thing, *carefully made*

**2. Plain noun, italic qualifier**
> the morning brief, *quietly*

**3. Imperative, low key**
> begin, *when you're ready*

Don't write headlines that try to be funny. Don't write headlines that try to sell.

## Microcopy patterns

**Buttons / actions** (always lowercase, plain verb):
- `open the docs →`
- `try it`
- `read on`
- `keep going`
- `bring it back`

Never: `Get Started Now!`, `Try It Free!`, `Learn More`.

**Empty states**:
- headline: short, with one italic
- subtext: one sentence on what would fill this space
- action: a text link, not a button

Example:
> nothing here yet, *just begun*
>
> the first build will appear once you push to main.
>
> `read the docs →`

**Errors**:
- Calm. State what happened. Offer one action.
- `something didn't load. retry?`
- Never: `Oops! Something went wrong! 😢`

**Loading**:
- The breathing ring. Optionally one italic phrase below it.
- `still listening`
- `just a moment`

**Confirmations**:
- `done` · `saved` · `sent` · `it's there`
- Never: `Success! 🎉`

## Tone stress tests

Read these out loud. Do they sound like Aura?

✅ "the morning brief, quietly"
❌ "Your AM Update Is Here! 🌅"

✅ "begin when you're ready"
❌ "Get Started Now!"

✅ "we'll try the next one"
❌ "Don't worry, we got you!"

✅ "nothing here yet, just begun"
❌ "It's empty in here! Add something to get started!"

If your copy sounds like the right column, rewrite it.

---

## When to break the rules

Two cases:

1. **Legal / regulatory copy** — must be exact. Capitalize what the lawyer says to capitalize.
2. **Direct user-authored content** — usernames, project titles, comments. Render as-is.

That's it. Everything else, follow the spec.
